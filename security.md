# Security Policy

## Supported Version

This repository is intended for learning and educational purposes. The latest version available in the repository should always be considered the supported version.

---

## Reporting a Security Issue

If you discover a security issue related to this repository or its documentation, please open an issue with the following information:

- A brief description of the issue
- Steps to reproduce the issue (if applicable)
- Expected behavior
- Actual behavior
- Suggested solution (optional)

Please avoid including sensitive information such as passwords, private keys, or personal data.

---

## Responsible Disclosure

If the reported issue could impact the security of users who may reuse the provided configurations or documentation, please report it responsibly and provide enough information for it to be reviewed.

---

## Disclaimer

This project was developed in a controlled lab environment for educational purposes. It should not be considered a production-ready deployment. Anyone using the configurations or procedures provided in this repository should review and adapt them according to their own environment before deployment.
