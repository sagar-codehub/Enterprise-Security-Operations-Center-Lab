# Contributing

Thank you for your interest in contributing to this project.

## How to Contribute

If you would like to improve this project, you can:

- Report issues or documentation errors.
- Suggest improvements to existing documentation.
- Add new detection scenarios.
- Improve configuration examples.
- Update MITRE ATT&CK mappings.
- Share ideas to improve the SOC lab.

## Pull Requests

Before submitting a pull request:

- Ensure the documentation is accurate.
- Keep the formatting consistent.
- Test any configuration changes before submitting them.
- Provide a clear description of the proposed changes.

## Questions

If you have any questions or suggestions, feel free to open an issue in this repository.
