# Repository Structure

This document provides a quick overview of the folders used in this project and their purpose.

```
Enterprise-Security-Operations-Center-SOC-Lab/
│
├── docs/
├── configs/
├── scripts/
├── README.md
├── LICENSE
├── CHANGELOG.md
├── CONTRIBUTING.md
├── SECURITY.md
└── CODE_OF_CONDUCT.md
```

## docs/

Contains all the project documentation, including architecture, Windows and Linux detections, threat hunting guides, MITRE ATT&CK mapping, troubleshooting, and investigation reports.

---

## configs/

Contains the configuration files used during the project, such as Wazuh, Auditd, Sysmon, SSH, and File Integrity Monitoring configurations.

---

## scripts/

Contains any helper scripts used to generate events, perform testing, or automate repetitive tasks during the lab.

---

## README.md

The main project documentation. It provides an overview of the lab, technologies used, detection scenarios, and links to the detailed documentation.

---

## LICENSE

Specifies the license under which this project is shared.

---

## CHANGELOG.md

Tracks major updates and improvements made to the project over time.

---

## CONTRIBUTING.md

Provides guidelines for anyone who wants to contribute or suggest improvements to the project.

---

## SECURITY.md

Explains how to report security-related issues or vulnerabilities found in this repository.

---

## CODE_OF_CONDUCT.md

Defines the expected behavior for contributors and helps maintain a respectful community.
